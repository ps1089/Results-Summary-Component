# Results summary component solution

This is a solution to the [Results summary component challenge on Frontend Mentor](https://www.frontendmentor.io/challenges/results-summary-component-CE_K6s0maV).

# Overview

A result card component design that is simple and responsive.

# Built Using

- HTML
- SCSS
- JS

# Process

The interactive and responsive card component was developed using the following process.
- HTML,an effort was made to use semantically relevant tags.
- Basic CSS,CSS flexbox and media queries were utilized.
- SCSS was used to optimize the stylesheets.
- Using JS,JSON file was read and DOM was utilized to render the contents.
